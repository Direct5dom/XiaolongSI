Windows Shell美化：PowerShell的美化

Posted on 2021-10-03 Edited on 2021-10-04 In Windows Terminal

PowerShell也可以很好看！

在之前我将Linux的Shell换成zsh，并通过oh-my-zsh对其进行了美化（然而最后把kali的zsh模板搬出来了，我还挺喜欢kali的这个样子的）。

美化完Linux的Shell后，面对Windows上的PowerShell越看越丑，于是便想着也能美化一下就好了。

PowerShell的性质和zsh本身一样，都是强大的Shell，并且可以定制化。但正如我们美化zsh使用oh-my-zsh一样，PowerShell同样有这种简便的美化工具，其中之一就是大名鼎鼎的oh-my-posh。

oh-my-posh是由Jan De Dobbeleer和众多社区贡献者所创造的。

本文主要写Windows下的PowerShell美化，并且默认终端为Windows Terminal
