完整的树莓派的入门指南

Posted on 2021-09-08 Edited on 2021-10-31 In 树莓派

Before the Start

为什么选择树莓派？

这节是为没做好选择的人准备的。如果你已经明确了自己要选择树莓派，请跳过这一节。

树莓派是什么？按照官方的说法：

Raspberry Pi is a series of small single-board computers (SBCs) developed in the United Kingdom by the Raspberry Pi Foundation in association with Broadcom.

这里明确了树莓派的定位：Small Single-board Computer (SBC)，也就是微型单板计算机。

因此要明确的是，树莓派并非单片机 (Microcontroller, MCU)，而是一种比单片机更“高级”的计算机。

一个不负责任但正确的粗俗说法：越”高级”意味着越靠近软件，越“低级”意味着越接近硬件。

如果你想要玩硬件，做避障小车，搞嵌入式开发，那么更加靠近硬件的单片机更加适合你。

如果你想要玩软件，架设私有NAS，搞长期爬虫，那么更加靠近软件的树莓派更加适合你。

为什么是树莓派？

这节是为新人准备的。如果你已经是个老手了，请跳过这节。

在微型SBC领域，或者换到开发板领域，树莓派并非独一家。有很多其他的厂商也在做，它们的价格或许更便宜，性能也许更强，那么为什么不选择别的板子呢？

原因在于，树莓派作为最知名的微型SBC开发板，他的资源是最全的。

这就好比一款手机APP，可以没有WP版，但是必须要有Android和iOS版，而你几乎无法在Ubuntu touch上找到现在的主流软件。

开发板也一样，其他的板子或许有各种优点，但是可能存在一系列未知或者没有良好解决办法的Bug，作为新人暂时没有解决经验和技术产出能力，这些杂七杂八的事情很容易干扰心情。因此树莓派是最好的选择。

这节是为没有搞清树莓派版本的人准备的。如果你能分清，请跳过这节。

描述 最通常意义上的树莓派，也是型号最多的一个系列。 最低成本的树莓派系列，适合IoT开发使用。 树莓派的单片机芯片 树莓派的单片机 更加完整的树莓派，更接近个人电脑。

树莓派现在也在出单片机芯片和开发板，但我们本文所探讨的树莓派均指代非单片机版本。因此可选项只剩下普通版、Zero系列、树莓派400。

这三者的性能排序（只考虑最新型号）：

树莓派400 > 普通版 > Zero

价格和上面的排序也是相同的（普通版4B 8GB版本的价格超过了树莓派400），毕竟一分钱一分货。

选择哪个树莓派？

这节是为没有选择好树莓派版本的人准备的。如果你已经选择好，请跳过这节。

本文不讨论单片机版本的树莓派，因此我们首先排除了Pico和他的芯片RP2040

对于剩下的三个版本的选择。

Zero系列

Zero版因为性能问题，它在软件方面能做的事情很少，它最常作为一个IoT开发或部署设备出现，在树莓派SBC领域算是靠近硬件应用的一个产品，因此如果你打算制作硬件领域的应用，可以选择Zero系列。

同样因为性能问题，无法支持起图形界面的使用，你的一切操作都只能局限在Linux命令行下操作，如果你不熟悉在Linux命令行下的操作，使用Zero将会是一场噩梦。

注意：这里必须明确的是，树莓派的很多硬件项目虽然看起来只是硬件的调用，但背后可能存在大量的复杂算法，而这种项目Zero可能难以承受。

Zero有两个版本：

  • Zero：最早的Zero版本，不带WLAN和蓝牙模块。
  • Zero W(ireless)：带WLAN和蓝牙模块的Zero。

硬件规格：

  • Zero：

    • 1GHz single-core CPU
    • 512MB RAM
    • Mini HDMI port
    • Micro USB OTG port
    • Micro USB power
    • HAT-compatible 40-pin header
    • Composite video and reset headers
    • CSI camera connector (v1.3 only)

  • Zero W

    • 1GHz single-core CPU
    • 512MB RAM
    • Mini HDMI port
    • Micro USB OTG port
    • Micro USB power
    • HAT-compatible 40-pin header
    • Composite video and reset headers
    • CSI camera connector (v1.3 only)

普通版

普通版树莓派是树莓派最具代表性的一个系列，一般提到树莓派说的就算这个系列。

这个版本的树莓派也是领域最全面的，也是最适合新人入手的系列。因为型号众多，版本众多，这里只探讨最新的4B版。

树莓派4B的性能也是树莓派系列第一次支撑得起流畅的图形界面的版本，因此是入坑的最佳推荐。假如你不熟悉命令行的操作，还可以打开图形界面去寻找问题的原因，对于没有命令行经验的新手来说是很好的选择。

树莓派4B有三个版本，对应内存为2GB、4GB以及8GB，内存越大价格越贵。在内存的选择上没有什么最好与最坏之分，需求仍然决定一切，对于没有明确自己需求的人来说，4GB版是最好的选择。

2GB的版本不适合图形界面的使用，而对于纯新人来说图形界面还是有必要存在的。8GB是为64位操作系统所准备的，这个版本的系统目前Bug仍比较多（除非你选择其他操作系统入坑，但不建议）。

4GB版本兼顾了价格和性能，同时4GB也作为64位操作系统的准入门槛，在未来或者当下即可尝试64位操作系统。

树莓派4B的硬件规格：

  • Broadcom BCM2711, Quad core Cortex-A72 (ARM v8) 64-bit SoC @ 1.5GHz
  • 2GB, 4GB or 8GB LPDDR4-3200 SDRAM (depending on model)
  • 2.4 GHz and 5.0 GHz IEEE 802.11ac wireless, Bluetooth 5.0, BLE
  • Gigabit Ethernet
  • 2 USB 3.0 ports; 2 USB 2.0 ports.
  • Raspberry Pi standard 40 pin GPIO header (fully backwards compatible with previous boards)
  • 2 × micro-HDMI ports (up to 4kp60 supported)
  • 2-lane MIPI DSI display port
  • 2-lane MIPI CSI camera port
  • 4-pole stereo audio and composite video port
  • H.265 (4kp60 decode), H264 (1080p60 decode, 1080p30 encode)
  • OpenGL ES 3.1, Vulkan 1.0
  • Micro-SD card slot for loading operating system and data storage
  • 5V DC via USB-C connector (minimum 3A*)
  • 5V DC via GPIO header (minimum 3A*)
  • Power over Ethernet (PoE) enabled (requires separate PoE HAT)
  • Operating temperature: 0 – 50 degrees C ambient

* A good quality 2.5A power supply can be used if downstream USB peripherals consume less than 500mA in total.

树莓派400

是一个键盘计算机，在上个世纪这种形式的计算机很多。

个人认为这个版本的树莓派才算完成树莓派最开始的任务：

The Raspberry Pi project originally leaned towards the promotion of teaching basic computer science in schools and in developing countries.

Raspberry Pi 项目最初倾向于在学校和发展中国家促进基础计算机科学的教学。

作为一个整合度较高的树莓派，它的任务几乎被局限在软件领域（虽然提供了GPIO结构，但是因为结构问题并不适合硬件应用）。而它确实也拥有全系列最高的配置，对于软件上的应用，无论是人脸识别，机器学习这种算法，还是搭中控、NAS，都是一个很好的选择。

不过按照设计，树莓派400更推荐的用法还是当成一台个人电脑来用，但这又是一个仁者见仁的东西。

树莓派400的硬件规格：

  • Broadcom BCM2711 quad-core Cortex-A72 (ARM v8) 64-bit SoC @ 1.8GHz
  • 4GB LPDDR4-3200
  • Dual-band (2.4GHz and 5.0GHz) IEEE 802.11b/g/n/ac wireless LAN
  • Bluetooth 5.0, BLE
  • Gigabit Ethernet
  • 2 × USB 3.0 and 1 × USB 2.0 ports
  • Horizontal 40-pin GPIO header
  • 2 × micro HDMI ports (supports up to 4Kp60)
  • H.265 (4Kp60 decode); H.264 (1080p60 decode, 1080p30 encode); OpenGL ES 3.1, Vulkan 1.0
  • MicroSD card slot for operating system and data storage
  • 78- or 79-key compact keyboard (depending on regional variant)
  • 5V DC via USB connector
  • Operating temperature: 0°C to +50°C ambient
  • Maximum dimensions 286 mm × 122 mm × 23 mm

其他需要知道的

你现在已经明白了什么是树莓派、为什么选择树莓派、树莓派的种类并且选择好了自己想要的树莓派，但是先不要着急，你还需要知道一些其他的内容。

Linux 不是Windows

在树莓派上最好的入门操作系统是Linux，或者更准确的说是Raspberry Pi OS，一个基于Debian的Linux发行版。

操作系统的差异意味着你很难将常见的软件运行在树莓派上，日常软件如QQ、微信；商业软件如Office、Chrome (有开源社区版)、Edge。

Linux的生态从日常使用的PC角度来讲，和Windows的差别简直比太阳到火星的距离还要大。只有在服务器领域、编程领域的生态能和Linux打平，只有在一些创客的作品或者小圈子里才能看到Linux Only的软件或程序。

ARM 不是x86

上面提到了Linux在日常领域不如Windows，这也许会让你萌生在树莓派上安装Windows的想法。这的确是可行的，但是这里就要提到第二个差异：架构差异。

树莓派使用的是ARM架构，而不是x86。而架构的差异最先体现在x86下很多常见的软件并不能运行，因此即便你安装了ARM版Windows，你同样不能运行你想运行的很多Windows程序，而且因为ARM Windows起步更晚，你在ARM Windows下的体验甚至不如Linux。

综上，如果你抱着任何日常使用树莓派替代主力PC的想法，请你马上打消掉。

Start

购买

选择版本购买

搞清楚须知并选择好树莓派的版本后，你要做的第一件事就是下单树莓派。

你可以在树莓派官网找到Buy a Raspberry Pi的页面，在这里你可以选择自己想要的树莓派，点击进入详情。

也许你会害怕从这里购买会导致跨境消费和跨境快递等一系列问题，这你不用担心，在你点击Buy Now后，会跳转到代理商选择页面，这里的国家会根据IP选择，你也可以手动选择为China。你会看到好几家可以购买的店铺，其中大部分在淘宝，我们也只需要通过淘宝购买就可以了。

选择配件或套装

树莓派除了单板以外，还需要一些其他配件才能正常启动运行。这当中两个必需品就是：

  • 电源
  • SD卡（注意：SD卡需要连接到Windows PC进行一些操作，若电脑没有SD卡接口，需要自行准备读卡器或大卡卡套）

因此最好一同下单这些东西，当然很多店家也会选配好一些套装，这当中可能会带有上面提到的读卡器、大卡卡套，也会有外壳。这里尤其建议买外壳，因为树莓派的针脚外露，板子本身也比较脆弱，需要做好保护。建议的组合是：

  • 树莓派板子
  • 外壳
  • 风扇
  • SD卡
  • 电源线
  • 大卡卡套/读卡器

只需要挑选和上述组合近似的套装即可。

此外关于一些“入门者套装”或者“传感器”，即给了一大堆传感器或者键盘鼠标的套装，个人建议不要购买。

对于键盘鼠标，我们日后几乎不会将其接在树莓派上使用，对于传感器，我们只需要日后根据个人需要选购即可，大部分传感器套装的传感器可能一辈子也用不上。

准备

操作系统镜像

我们需要先给树莓派安装系统，和PC不同，树莓派的系统安装是.img格式文件烧录的做法。

首先你需要一个烧录工具，这里推荐：Win32 Disk Imager

接下来你需要下载系统镜像，虽然你可以选择从官网下载，或者使用官方的边下边装镜像，但是因为网络问题这里还是建议使用国内镜像来下载。

这里推荐清华大学开源软件镜像站，你可以找到Index of /raspberry-pi-os-images/，在里面找到你需要的操作系统版本，下载镜像即可。

下载好的镜像一般是.zip格式的压缩包，你需要解压缩才能看到.img的镜像文件。
